Odin Recipes

This project is part of the The Odin Project Foundations curriculum. The goal of the project is to practice basic HTML skills, file structure organization, and linking between web pages by building a simple recipe website.

Project Overview

The Odin Recipes project is a basic website that displays several recipes. The website consists of a main homepage and multiple recipe pages. Each recipe page contains an image of the dish, a description, a list of ingredients, and step-by-step instructions on how to prepare the meal.

This project focuses on practicing:

Basic HTML structure

Creating and organizing directories

Linking pages using relative paths

Using lists and images in HTML

Building a simple multi-page website
